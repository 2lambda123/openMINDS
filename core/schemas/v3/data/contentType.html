ContentType (core - v3)

https://openminds.ebrains.eu/core/ContentType - JSON Schema

associatedFileExtension array Enter one or several file extensions associated with this content type.
description string Enter a description of the content type specification. May be left blank if a public specification can be linked in 'specification'.
relatedMediaType string (iri) Enter the iternationalized resource identifier (IRI) of the official registered media type (e.g. on IANA.org) matching this content type.
name string Enter the name (iana-inspired convention) of this content type.
specification string (iri) Enter the iternationalized resource identifier (IRI) of the official specification of this content type. Leave blank and use 'description' to provide some specification if an official specification is not available.
synonym array Enter one or several synonyms of this content type.
